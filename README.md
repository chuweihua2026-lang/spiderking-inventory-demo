# 蜘蛛王多门店经营中台演示版

这是蜘蛛王多门店进销存、门店销售、会员 CRM、库存预警的一版可交付 Web 演示原型。

当前版本定位：

- 用于第一版业务演示
- 前端内置模拟数据
- 不依赖数据库
- 不做真实登录
- 可点击核心流程
- 后续可扩展为 React + API + PostgreSQL + Prisma 的正式系统

## 已覆盖内容

- 经营驾驶舱
- 商品/SKU 管理
- 总部库存总览
- 同一 SKU 的 5 家门店横向库存对比
- 入库演示
- 门店间调拨演示
- 多商品小票式收银
- 退货回可售库存或残次库存
- 门店盘点草稿与提交调整
- 会员 CRM、积分、储值、邀约
- 库存分析和完整预警入口
- 角色权限展示和基础字段权限

## 演示门店

- 蜘蛛王一店
- 蜘蛛王二店
- 邦赛一店
- 邦赛二店
- 澳伦总店

## 核心规则

- SKU 是最小库存单位：货号 + 颜色 + 尺码
- 每个门店都有独立库存
- 总部可按门店、品牌、SKU 查看库存
- 销售允许负库存，负库存会进入预警
- 退货可进入可销售库存或残次库存
- 盘点支持草稿，提交后调整库存
- 积分按实付金额 1 元 = 1 分
- 储值支持充值和消费扣减的演示结构

## 本地启动

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

构建检查：

```bash
npm run build
```
